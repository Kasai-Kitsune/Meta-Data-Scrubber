How it works

Images (incl. HEIC): visible pixels are decoded and written to a new image. HEIC is converted to PNG (preserves transparency). EXIF and similar metadata are not intentionally copied.

Standard audio/video: FFmpeg rewrites the container, maps normal media streams, and omits global metadata, chapters, subtitles, attachments, cover art, and data streams.

Maximum audio/video: normal media streams are decoded and re-encoded into the same general file type while those extra streams and metadata are omitted.

Documents: For PDF, author/title/etc. are cleared. Office/ODF files are unzipped and metadata XML files are removed, then repacked.

This tool does not claim universal removal of every unknown proprietary structure. It reports the operation it actually performed.
